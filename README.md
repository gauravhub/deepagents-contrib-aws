# deepagents-contrib-aws

AWS contributions for Deep Agents.

## Installation

```bash
pip install deepagents-contrib-aws
```

## Development

```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create venv and install dependencies
uv sync

# Run tests
uv run pytest

# Lint
uv run ruff check .

# Build
uv build

# Publish to PyPI
uv publish
```
